MediaTemple (dv) Backup to S3 Script

Fri 18 December 2009

In Code.

## The Problem MediaTemple servers run the *Plesk Control Panel*, which *has* a reputation for having *useless backups*. ## The Solution * MySQL Dumps of all Databases and Tables * All configured vhosts, zipped up * Pushes it all to either S3 or FTP Stick it in /etc/cron.daily/, and you'll be good to go.
